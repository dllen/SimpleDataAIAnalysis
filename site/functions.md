# 系统功能说明

本文档用于介绍 Data Analysis Agent 的系统功能与说明，便于团队、用户或文档阅读者快速理解产品定位、核心能力、安全边界与后续演进方向。

## 1. 产品定位

Data Analysis Agent 是一个面向非专业用户的个人数据问答助手。用户上传表格数据后，可直接用自然语言获得分析结论、图表和可执行建议，而不是先学习 SQL、透视表或 BI 工具。

## 2. 核心功能

### 2.1 上传与识别

- 支持 CSV、Excel、JSON 等常见表格格式。
- 自动识别字段名、字段类型和基础数据质量。
- 为后续分析、清洗和可视化建立可执行的数据结构。

### 2.2 自然语言分析

- 用户用中文或其他自然语言提问。
- 系统基于 LLM 生成 SQL，由 DuckDB 执行。
- 再把查询结果翻译成自然语言结论，而不是只返回原始表格。

### 2.3 数据清洗

- 自动扫描缺失值、异常值、重复数据和列名问题。
- 生成清洗建议，用户可以确认、编辑或取消。
- 清洗结果可保存为新数据集，也可覆盖原数据，并支持回滚。

### 2.4 可视化

- 自动生成图表，例如柱状图、饼图等。
- 帮助用户快速理解数据分布、趋势和异常点。
- 结果可与分析结论共同展示，减少用户二次加工成本。

### 2.5 多轮对话

- 支持上下文连续分析。
- 用户可以在上一轮结论基础上继续追问，获得更深层结果。

### 2.6 安全与隔离

- 采用 JWT 认证和用户数据隔离。
- 分析执行阶段有 SQL 只读沙箱约束。
- 清洗操作支持备份与回滚，降低误操作风险。

## 3. 系统边界与说明

- 当前产品重点解决“个人轻量自助分析”，不默认面向复杂企业级报表体系。
- 分析结果依赖模型输出质量，因此系统会配合推荐追问、错误提示和人工可干预能力持续优化体验。
- 系统功能会随 SaaS 演进持续扩展，当前版本优先保障核心上传、分析、清洗、可视化闭环。

## 4. 后续方向

- 游客演示体验
- 自动推荐问题
- AI 结论摘要
- 场景模板库
- 结果导出与分享
- 运营看板与转化优化
